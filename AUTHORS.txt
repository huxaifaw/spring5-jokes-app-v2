		@RequestMapping is one of the most important annotations of spring.
		It is used to map incoming HTTP requests to handler method of MVC and REST Controllers.
		When configuring Spring MVC, one should provide the mapping between Web Requests and Handler methods.
		Mapping can be provided at Method level and Class level depending on your use case.
		The annotated method returns a String which the view name that is to be displayed on screen when specific request in @RequestMapping is encountered.
